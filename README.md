# DG LAUNCHER

DG LAUNCHER — это мобильный лаунчер для iPhone и других смартфонов, стилизованный под приложение с нижней навигацией и поддержкой персонализации интерфейса.

## Возможности

- мобильный интерфейс с адаптацией под iPhone
- вкладки:
  - Магазин
  - Библиотека
  - Настройки

## Структура приложения

### Магазин
Главная страница лаунчера. Сейчас используется как стартовый экран и информационный раздел.

### Библиотека
Содержит доступные игры пользователя.  

### Настройки
Позволяют:
- менять тему
- включать или выключать звук
- менять фон приложения
- выбирать аватарку профиля

## Аватарка профиля

При первом запуске лаунчер предлагает выбрать аватарку.  
Она отображается справа сверху в интерфейсе.

Доступно:
- выбор встроенных вариантов
- загрузка собственной картинки

## Фон приложения

В настройках можно выбрать один из вариантов оформления:

- стандартный
- анимированный градиент
- собственное фото из галереи

## Сохранение данных

Приложение сохраняет пользовательские настройки в `localStorage`, поэтому после перезапуска остаются:

- выбранная тема
- состояние звука
- аватарка
- фон

## Запуск

Открой HTML-файл в браузере или подключи его к своему сайту/веб-оболочке.

Если приложение используется как Web Clip на iPhone, оно может открываться почти как отдельное приложение с домашнего экрана.

## Используемые технологии

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- `localStorage`
- `AudioContext` для звуков интерфейса

## Примечание

Некоторые функции, зависящие от браузера или iOS, могут работать по-разному на разных устройствах.  
Например:
- звук может требовать первое касание пользователя
- вибрация может не поддерживаться в Safari/Web Clip
